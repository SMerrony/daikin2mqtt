# Building daikin2mqtt

## For Production Use...
`gprbuild -Xmode=release`

## Prerequisites
* `ada-toml` from https://github.com/pmderodat/ada-toml
* MQTT-related...
  * `libmosquitto-dev`
  * `mosquitto` (mosquitto-ada) from https://github.com/persan/mosquitto-ada
* `libaws19-dev` (or later)
* `libgnatcoll-dev`
